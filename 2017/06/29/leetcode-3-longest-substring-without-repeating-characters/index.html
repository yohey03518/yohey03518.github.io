[LeetCode] 3. Longest Substring Without Repeating Characters

今天來做的是Algorithms的第三題，難度為Medium。

題目要求是傳入一個字串，要回傳最大長度的不重複字元的子字串(Substring)的長度，

(題目連結)

官方範例：

Examples:

Given "abcabcbb", the answer is "abc", which the length is 3.

Given "bbbbb", the answer is "b", with the length of 1.

Given "pwwkew", the answer is "wke", with the length of 3. Note that the answer must be a substring, "pwke" is a subsequence and not a substring.

以下是思考許久後的撰寫的結果：

https://gist.github.com/yohey03518/1b85b1f466d3150d1a39be213e130fa9

想法是走訪字元的過程中，去記錄目前正在try最長長度的各字元所在陣列的索引值，

當遇到重複的值時，除計算目前長度大小外，

便從重複字元的下一個字元開始重新走訪，

例如abcecp，當從a開始走訪走到第二個c時，

除記錄目前長度外(目前為4)，便會從第一個c的下一個字元(e)重啟走訪。

 

好的那解釋完畢，測試也通過了，趕緊信心滿滿的submit看看
